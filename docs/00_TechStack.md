# Tech Stack Documentation

## Core Framework
- **Runtime:** React 18
- **Build Tool:** Vite (Fast, ESM-based build tool)
- **Language:** TypeScript

## Styling & UI
- **CSS Framework:** Tailwind CSS
- **Animation:** Framer Motion, GSAP (GreenSock)
- **3D/Visuals:** React Three Fiber (@react-three/fiber), Drei, OGL
- **Icons:** Lucide React

## State Management & Logic
- **Routing:** React Router (implicit/manual currently, migrating to structured routing if needed)
- **State:** React Hooks (useState, useEffect)

## Build & Deploy
- **Package Manager:** npm
- **Deployment:** Vercel (Recommended)

## Deviation from Master Framework
*Note: The Master Framework recommends Next.js. This project uses Vite. All architectural decisions in this project should respect the Client-Side Rendering (CSR) nature of Vite, unless SSR is specifically introduced.*
