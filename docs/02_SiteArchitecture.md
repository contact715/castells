# Site Architecture: Castells

## 1. Navigation Structure

### Main Navigation (Desktop)
- **Work** (Anchor to `#work`)
- **Services** (Anchor to `#services`)
- **Process** (Anchor to `#process`)
- **About** (Link to `/about`)
- **Careers** (Link to `/careers`)
- **CTA Button:** "Start Your Project" (Anchor to `#contact`)

### Mobile Menu
- Hamburger menu expanding to full screen.
- Contains same links as desktop.

## 2. Page Structure

### Home Page (`/`)
1.  **Hero Section:** Video background, headline, "Start Your Project" CTA.
2.  **Trusted By:** Logo loop of client partners.
3.  **Work:** Selected case studies (ScrollStack effect).
4.  **Industries:** Grid of industries served (SaaS, E-com, etc.).
5.  **Why Choose Us:** Value propositions.
6.  **Services:** Detailed service offerings (Paid Social, PPC, etc.).
7.  **Process:** Step-by-step workflow (Discovery -> Strategy -> Execution -> Scale).
8.  **Team:** Founders/Key team members.
9.  **Growth Audit:** Interactive multi-step form for lead qualification.
10. **FAQ:** Accordion style frequently asked questions.
11. **Blog/Insights:** Featured articles.
12. **CTA:** Final call to action.
13. **Footer:** Links, social media, copyright.

### Case Study Detail (`/case-study`)
- **Hero:** Project title, client name, key metrics (ROAS, Revenue).
- **Challenge:** Problem statement.
- **Solution:** Strategy and execution details.
- **Results:** Quantitative outcomes.
- **Visuals:** High-quality images/videos of the work.
- **Next Project:** Link to next case study.

### About Page (`/about`)
- **Mission:** "The Anti-Agency" philosophy.
- **Story:** History of Castells.
- **Values:** Core principles.
- **Team:** Extended team bios.

### Careers Page (`/careers`)
- **Culture:** Working at Castells.
- **Open Positions:** List of current job openings.
- **Application:** Link to apply.

## 3. URL Structure
- `/` - Home
- `/about` - About Us
- `/careers` - Careers
- `/case-study` - Case Study Details (Dynamic content based on selection)
